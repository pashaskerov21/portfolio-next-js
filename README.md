# Alipasha Askerov — Personal Portfolio 🚀

A personal developer portfolio built in **three versions** using three different modern frontend frameworks — all sharing the same design system, content structure, and TypeScript codebase. This demonstrates framework-agnostic frontend development skills and deep understanding of each ecosystem.

---

## 🌐 Live Demos

| Version | Framework | Demo | Repository |
|---------|-----------|------|------------|
| ⭐ **Primary** | Next.js 14 | [alipashaskerov.vercel.app](https://alipashaskerov.vercel.app) | [portfolio-next-js](https://github.com/pashaskerov21/portfolio-next-js) |
| Angular | Angular 18 | [alipashaskerov2.vercel.app](https://alipashaskerov2.vercel.app) | [portfolio-angular](https://github.com/pashaskerov21/portfolio-angular) |
| Vue | Nuxt.js 3 (Vue 3) | [alipashaskerov3.vercel.app](https://alipashaskerov3.vercel.app) | [portfolio-vue](https://github.com/pashaskerov21/portfolio-vue) |

> ⭐ The **Next.js version** is the primary and most feature-complete implementation.

## 🏗️ Version Breakdown

### ⭐ Version 1 — Next.js 14 (Primary)

The main version. Built with the latest Next.js App Router, SSR/SSG, and React ecosystem tools.

**Tech Stack:**

![Next.js](https://img.shields.io/badge/Next.js_14-000000?style=flat-square&logo=nextdotjs&logoColor=white)
![React](https://img.shields.io/badge/React_18-61DAFB?style=flat-square&logo=react&logoColor=black)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)
![SCSS](https://img.shields.io/badge/SCSS-CC6699?style=flat-square&logo=sass&logoColor=white)

**Key Features:**
- App Router with file-based routing
- SSR & SSG for optimal performance and SEO
- React Hooks for state and lifecycle management
- ESLint configured for code quality

**Project Structure:**
```
portfolio-next-js/
├── app/              # Next.js App Router pages & layouts
├── src/
│   ├── components/   # Reusable React components
│   └── types/        # TypeScript type definitions
└── public/           # Static assets
```

### Version 3 — Nuxt.js 3 (Vue 3)

Built with Nuxt.js 3 on top of Vue 3, using the Composition API, composables, and Nuxt's file-based routing.

**Tech Stack:**

![Nuxt.js](https://img.shields.io/badge/Nuxt.js_3-00DC82?style=flat-square&logo=nuxtdotjs&logoColor=white)
![Vue.js](https://img.shields.io/badge/Vue_3-4FC08D?style=flat-square&logo=vuedotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)
![SCSS](https://img.shields.io/badge/SCSS-CC6699?style=flat-square&logo=sass&logoColor=white)

**Key Features:**
- Vue 3 Composition API
- Nuxt auto-imports and composables
- File-based routing via Nuxt conventions
- SSR support via Nuxt's rendering engine
- Reusable composables in `composables/` directory

**Project Structure:**
```
portfolio-vue/
├── app/              # Nuxt app directory
├── composables/      # Vue composables (shared logic)
├── types/            # TypeScript type definitions
└── public/           # Static assets
```

## 🚀 Getting Started

### Next.js Version

```bash
git clone https://github.com/pashaskerov21/portfolio-next-js.git
cd portfolio-next-js
npm install
npm run dev
```

### Angular Version

```bash
git clone https://github.com/pashaskerov21/portfolio-angular.git
cd portfolio-angular
npm install
ng serve
```

### Vue / Nuxt.js Version

```bash
git clone https://github.com/pashaskerov21/portfolio-vue.git
cd portfolio-vue
npm install
npm run dev
```
